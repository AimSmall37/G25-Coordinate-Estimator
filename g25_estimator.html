G25 Coordinate Estimator

Approximate PCA coordinates from known inter-sample distances via weighted multilateration
Method: Given distances from an unknown sample to N reference samples with known G25 coordinates, this tool estimates the unknown sample's position using inverse-distance-weighted centroid as an initial guess, then refines via gradient descent. When dimension weighting is enabled, a second pass blends per-dimension nearest-neighbor estimates with the multilateration result, then a final constrained polish preserves early-dim accuracy while restoring overall distance fit.
